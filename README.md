**备注：**  
- **MSE (Mean Squared Error)**：均方误差，越小越好  
- **MAE (Mean Absolute Error)**：平均绝对误差，越小越好  
- **R² (R-Squared)**：决定系数，越接近 1 越好  
- **计算时间**：推理时间，越小越好  
- **我们的模型 (MWCN)** 在多个指标上优于其他模型
